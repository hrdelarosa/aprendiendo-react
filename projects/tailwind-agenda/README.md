# Tailwind Agenda

## Descripción
Una aplicación de agenda que utiliza Tailwind CSS para el diseño.

## Tecnologías utilizadas
- React
- Vite
- Tailwind CSS
- Node.js (para la API)

  assets/

  components/
    Data.jsx
